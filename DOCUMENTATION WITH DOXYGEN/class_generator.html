Detailed Description

Building the tree is done recursively, by reading each line from the file, that line is processed and from it creates one of the following objects: Tag, plain text to a given element, Tag of type leaf. Tree elements are described with an XMLElement class that contains the name of the tag, a list of the tag's attributes, the text in this tag and a list of descendants(nested elements). Each element has a unique id. The bottom of the recursive creation of the tree is when a Tag of type leaf (ie an element that has no nested elements) is reached.
