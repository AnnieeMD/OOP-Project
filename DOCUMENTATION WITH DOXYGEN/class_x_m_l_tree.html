◆ UniqueID()

void XMLTree::UniqueID ( )

Generates unique IDs for each element in the XMLTree.

If the element has an “id” field and its value is unique for all elements in the file, this value is used. If the element has an "id" field, but its value is not unique, then a character string is concatenated to this id, which makes it unique. If the element does not have an “id” field, a unique identifier generated by the program is appended to it.

◆ xpath()

void XMLTree::xpath ( ) const

Executes an XPath query on the XMLTree.

This function displays the supported XPath queries and prompts the user to choose a query number. It then takes the user's input for the query and executes it on the XMLTree's root element.
